Removing Format

This website contains links to software which is either no longer maintained or will be supported only until the end of 2019 (CKFinder 2). For the latest documentation about current CKSource projects, including software like CKEditor 4/CKEditor 5, CKFinder 3, Cloud Services, Letters, Accessibility Checker, please visit the new documentation website.

CKEditor makes it easy to not only change the style of your text, but it also makes it equally easy to remove these styles.

To remove text styling applied to the document text, select the text fragment and press the toolbar button. All text styling will be removed and the text will be displayed in the default font.

The figure below shows a sample text with some styling — typeface, font size, italics, colors — applied.

Sample text formatted with CKEditor styles


After you select the text and use the Remove Format function, all text styling is gone and the text is displayed with default paragraph formatting.

Sample text with styles removed


Note that this applies to style properties that are applied to the HTML source code by using the CSS stylesheet rules. This category includes most text styling methods as described in the Text Styling section (without paragraph format) as well as text alignment and indentation.
